# GitHub PM Demo

Ovo je demonstracijski projekt za učenje upravljanja GitHub repozitorijima i projektima kroz CLI.

## Struktura Projekta

- `src/`: Izvorni kod aplikacije.
- `tests/`: Unit testovi.
- `.gitignore`: Pravila za ignoriranje datoteka.

## Kako pokrenuti

1. Klonirajte repozitorij.
2. Pokrenite `python src/main.py`.
