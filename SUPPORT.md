# Support

## Getting Help

If you have questions about how to use this project, please:

1.  Read the [README](../README.md)
2.  Check the [Wiki](https://github.com/MarijanMiletic/github-pm-demo/wiki)
3.  Search existing [Discussions](https://github.com/MarijanMiletic/github-pm-demo/discussions)

## Reporting Bugs

Found a bug? Please open an issue using the [Bug Report Template](https://github.com/MarijanMiletic/github-pm-demo/issues/new?template=bug_report.md).

**Please do not use the issue tracker for general support questions.** Use Discussions instead.
